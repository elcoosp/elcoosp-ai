## 🚀 Getting Started

Each package is located in its own subdirectory and can be used independently.

### Prerequisites

- Node.js 18 or later
- pnpm (recommended) or npm/yarn

### Installation

For a specific package, see its own README. Usually you can install with:

```bash
npm install @elcoosp-ai/seedling
# or
pnpm add @elcoosp-ai/seedling
```

## 🛠 Development

This repository uses pnpm workspaces (if applicable) or simply contains multiple independent packages.

```bash
# Clone the repository
git clone https://github.com/elcoosp-ai/elcoosp-ai.git
cd elcoosp-ai

# Install dependencies for a specific package
cd seedling
pnpm install
```

Each package has its own scripts for building, testing, and linting – see the package’s `README.md` for details.
